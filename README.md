# fintech-reconciliation-engine

> Automated ledger reconciliation — compares transactions across two sources and generates audit reports.

## Stack
Python 3.11 · pandas · Jinja2 · Rich

## Quick Start

```bash
pip install -r requirements.txt
python -m recon run --source-a data/sample_source_a.csv --source-b data/sample_source_b.csv
```
